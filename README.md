# Isa Robertini Portfolio

A personal portfolio website showcasing the artworks, exhibitions, and web projects of Isa Robertini. Built with **React**, **React Router**, **Tailwind CSS**, and interactive components for a modern, responsive experience.

**Live Demo:** [https://isarobertini.com/](https://isarobertini.com/)

---

## Features

- **Portfolio Gallery**
  - Display of artworks, exhibitions, and websites.
  - Clickable images leading to detailed pages.

- **Navigation Menu**
  - Responsive, animated menu with light/dark background support.
  - Highlights active section and provides smooth transitions.

- **Artwork Detail Pages**
  - High-quality images with captions, dimensions, materials, and exhibition info.
  - Structured text descriptions and artist statements.

- **Interactive Components**
  - Fade-in animations using `react-awesome-reveal`.
  - Expandable sections for images and additional info.

- **Support & Credits**
  - Display of project sponsors and collaborators.
  - Links to supporting organizations (Konstnärsnämnden, Helge Ax:son Johnsons stiftelse).

- **Responsive Design**
  - Optimized for desktop, tablet, and mobile screens.
  - Adaptive layout with flexible image sizing.

---

## Usage

1. Explore artworks, exhibitions, and website projects.
2. Click images to view detailed project pages.
3. Read curated text descriptions and artist statements.
4. Access external links to supporting organizations and related content.
